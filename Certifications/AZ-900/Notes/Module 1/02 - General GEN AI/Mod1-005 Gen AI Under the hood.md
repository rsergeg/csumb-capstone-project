# Transformers

- major NLP breakthrough
- transformer architecture
- uses attention/self-attention

## Self-Attention

- AI checks importance of words
- context awareness
- understands relationships in text

## Important Difference

- processes multiple words at same time
- not strictly left-to-right
- faster + more efficient training

## BERT + GPT

- transformer-based models
- context understanding
- advanced language generation

---

# Machine Learning Types

## Supervised Learning

- labeled datasets
- correct outputs already known
- learns input → output mapping

### Example

- articles paired w/ summaries
- AI learns summary generation

## Unsupervised Learning

- no labels provided
- AI finds patterns independently
- more flexible/creative generation
- useful for adaptive systems

---

# Core Generative AI Technologies

## Neural Networks

- foundation of generative AI
- layers of connected nodes
- learns from large datasets

## GANs

- Generative Adversarial Networks
- generator vs discriminator
- one creates content
- other checks if realistic

## VAEs

- Variational Autoencoders
- compress + reconstruct data
- encoder + decoder system
- generates new samples from learned patterns
