
# Generative AI Terminology

---

# Artificial Intelligence (AI)

- computers doing human-like tasks
- reasoning
- learning
- problem solving
- language understanding
- decision making

## AI systems

- simple rule-based systems
- advanced neural networks
- adapts using data
- improves over time

## Machine Learning connection

- ML = major part of AI
- detects patterns in data
- predictions/probabilities
- learns from previous results

---

# Machine Learning (ML)

- AI systems learning from data
- no direct programming for every task
- predictions/decisions from patterns

## Types of Learning

### Supervised Learning

- labeled training data
- correct answers already known
- predicts future outcomes

### Unsupervised Learning

- no labeled data
- finds patterns/groups
- clustering relationships

### Reinforcement Learning

- learning through feedback
- rewards/penalties
- sequence-based decisions

## Common ML Algorithms

- linear regression
- decision trees
- neural networks

## Data Processing

- normalization
- handling missing values
- feature selection
- improves accuracy

## Performance Metrics

- accuracy
- precision
- recall
- ROC curve

## ML Uses

- finance/trading
- healthcare diagnostics
- self-driving vehicles

---

# Deep Learning (DL)

- advanced branch of ML
- uses deep neural networks
- many hidden layers
- learns from large datasets

## Main Feature

- automatic feature extraction
- recognizes complex patterns
- handles unstructured data

## Training Methods

- supervised
- semi-supervised
- unsupervised

## DL Applications

- image recognition
- NLP
- speech recognition

## Important Note

- DL helped advance generative AI
- supports advanced models like GANs

---

# Neural Networks (NN)

- inspired by human brain
- connected neurons/nodes
- pattern recognition system

## Structure

- layers of neurons
- weights + biases
- data passes through layers

## Learning Process

- adjusts weights over time
- improves predictions
- training through backpropagation

## Major Uses

- computer vision
- speech recognition
- NLP
- classification/prediction tasks

---

# Generative Adversarial Networks (GANs)

- two competing neural networks
- generator + discriminator

## Generator

- creates fake/generated data
- tries to look real

## Discriminator

- checks if data is real or fake
- evaluates generator output

## GAN Process

- competition improves results
- generator becomes more realistic
- discriminator becomes more accurate

## GAN Applications

- image generation
- video generation
- voice synthesis
- realistic AI media

---

# Natural Language Processing (NLP)

- AI understanding human language
- communication between humans/computers

## NLP Tasks

- reading text
- understanding language
- analyzing meaning
- generating responses

## NLP Techniques

- syntax parsing
- entity recognition
- sentiment analysis

## NLP Applications

- chatbots
- translation tools
- spam/email filtering
- voice assistants/GPS

---

# Transformers

- major advancement in DL + NLP
- introduced by Google (2017)
- "Attention is All You Need"

## Self-Attention

- AI checks importance of words
- context awareness
- understands relationships in sentences

## Important Difference

- processes words in parallel
- older systems processed sequentially
- faster + more efficient

## Transformer Uses

- translation
- summarization
- sentiment analysis
- image processing
- music generation

---

# GPT (Generative Pre-trained Transformers)

- OpenAI language models
- based on transformer architecture
- natural language generation

## Pre-Training

- trained on internet text
- unsupervised learning
- predicts next word/token

## Fine-Tuning

- adjusted for specific tasks
- translation
- summarization
- Q&A systems

## GPT Capabilities

- human-like responses
- conversational AI
- automated writing/content

## GPT-4

- 175 billion parameters
- larger/more advanced model

---

# Tokenization

- splitting text into tokens
- words/subwords/characters
- prepares text for AI processing

## Why Important

- standardizes input
- easier for models to process
- foundation of NLP systems

---

# Word2Vec

- converts words into vectors/numbers
- developed by Google
- shallow neural network method

## Purpose

- captures word relationships
- similarity between words
- context understanding

---

# BERT

- Bidirectional Encoder Representations from Transformers
- Google language model
- transformer-based system

## Main Advantage

- understands words using full context
- reads both directions in sentence
- stronger context awareness

## BERT Uses

- question answering
- sentiment analysis
- language understanding tasks
