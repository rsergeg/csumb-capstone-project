## What is Microsoft Azure?

- Microsoft's cloud computing platform
- Provides on-demand cloud services
- Lets businesses build, deploy, and manage applications
- Replaces or supplements traditional on-premise server rooms
- Scalable = resources can grow or shrink as needed

### Why Companies Use Azure

- Faster innovation
- Global availability
- Easier scaling
- Reduced hardware costs
- Supports remote and worldwide operations

---

# Core Azure Service Categories

The three major service categories:

1. Compute
2. Storage
3. Networking

Think:

**Compute = Processing**  
**Storage = Data**  
**Networking = Connectivity**

---

# Compute Services

### Compute

- Runs applications and workloads
- Processes data
- Provides computing power

Think:

- CPU
- Memory (RAM)
- Operating Systems
- Application execution

---

## Azure Virtual Machines (VMs)

- Virtual computers in Azure
- Can run Windows or Linux
- Similar to physical servers

Uses:

- Websites
- Business applications
- Databases
- Custom workloads

Exam Note:

- Azure VM = Infrastructure as a Service (IaaS)

---

## Azure App Service

- Platform for hosting web applications
- Supports multiple programming languages
- Azure manages the infrastructure

Benefits:

- Faster deployment
- Automatic scaling
- Less server management

Exam Note:

- App Service = Platform as a Service (PaaS)

---

## Azure Functions

- Serverless compute service
- Runs code when events occur
- No server management required

Examples:

- Process uploaded files
- Send notifications
- Trigger workflows

Exam Note:

- Pay only when code runs

---

## Azure Kubernetes Service (AKS)

- Managed Kubernetes platform
- Deploys and manages containers
- Azure handles Kubernetes infrastructure

Benefits:

- Easier container management
- Simplified scaling
- Reduced administration

---

## Azure Batch

- Large-scale computing jobs
- Parallel processing
- High Performance Computing (HPC)

Examples:

- Scientific simulations
- Video rendering
- Big data processing

---

# Storage Services

### Storage

- Stores data in Azure
- Highly scalable
- Accessible from anywhere

Think:

- Files
- Images
- Videos
- Databases
- Backups

---

## Azure Blob Storage

- Stores unstructured data

Examples:

- Images
- Videos
- Documents
- Backups
- IoT data

Benefits:

- Low cost
- Massive scalability

Exam Note:

- Blob = Binary Large Object

---

## Azure SQL Database

- Managed relational database
- Uses SQL structure
- Azure handles maintenance

Benefits:

- High availability
- Automatic updates
- Scalability

Think:

- Customer records
- Orders
- Business data

---

## Azure Files

- Managed network file shares
- Shared across multiple systems

Benefits:

- Easy file sharing
- Cloud-based access
- Highly available

---

## Azure Disk Storage

- Storage attached to Azure VMs
- Persistent storage

Uses:

- Operating systems
- Applications
- VM data

- Virtual hard drives

---

## Azure Data Lake Storage

- Designed for big data analytics

- Data science

# Networking Services

### Networking

- Connects Azure resources
- Controls traffic flow
- Improves security

Think:

- Highways connecting cloud resources

---

## Azure Virtual Network (VNet)

- Private network inside Azure
- Secure communication between resources

Benefits:

- Isolation from public internet
- Secure connectivity

Exam Note:

- VNet = foundation of Azure networking

---

## Azure Content Delivery Network (CDN)

- Delivers content closer to users
- Improves website speed

Examples:

- Images
- Videos
- Static website files

Benefits:

- Lower latency
- Faster load times

---

## Azure Load Balancer

- Distributes traffic across servers

Benefits:

- High availability
- Better performance
- Improved reliability

Think:

- Traffic cop directing requests

---

## Azure ExpressRoute

- Private connection to Azure
- Bypasses public internet

Benefits:

- More secure
- Predictable latency
- Higher reliability

Exam Note:

- ExpressRoute = private connectivity

---

## Azure Firewall

- Managed network security service
- Protects Azure networks

Features:

- Traffic filtering
- Threat protection
- High availability
