
# Azure Load Balancer Terms

You probably don't need deep notes on this section. It's mostly vocabulary for the next lesson.

## Important Terms

### OSI (Open Systems Interconnection)

- Networking model with 7 layers.
- Used to understand how network communication works.

---

### VM (Virtual Machine)

- Virtual computer running in Azure.
- Can run Windows or Linux.

---

### TCP (Transmission Control Protocol)

- Reliable network communication protocol.
- Ensures data arrives correctly.

Examples:

- Web browsing
- Email
- File transfers

---

### UDP (User Datagram Protocol)

- Faster but less reliable than TCP.
- Used when speed matters more than guaranteed delivery.

Examples:

- Streaming
- Online gaming
- Voice calls

---

### HTTP (Hypertext Transfer Protocol)

- Standard web traffic protocol.
- Used by websites.

Example:

- http://website.com

---

### HTTPS (Hypertext Transfer Protocol Secure)

- Secure version of HTTP.
- Encrypts communication.

Example:

- https://website.com

Exam Note:

- HTTPS is preferred because it is secure.

---

### NAT (Network Address Translation)

- Converts private IP addresses to public IP addresses.
- Allows devices to communicate with external networks.

---

### SHA256 (Secure Hash Algorithm 256)

- Cryptographic hashing algorithm.
- Used for security and data integrity.

---

### TLS (Transport Layer Security)

- Encrypts network communication.
- Used by HTTPS.

Purpose:

- Protect data in transit.

---

### HA (High Availability)

- Keeps applications available during failures.
- Minimizes downtime.

Examples:

- Availability Zones
- Load Balancers
- Traffic Manager

---

### NSG (Network Security Group)

- Azure firewall-like security rules.
- Controls network traffic.

Can allow or block:

- IP addresses
- Ports
- Protocols

Exam Note:

- NSGs are used to secure Azure networks.

---

### SKU (Stock Keeping Unit)

In Azure:

- Different service tiers or versions.
- Determines features and pricing.

Examples:

- Basic SKU
- Standard SKU

---

### IPv4

- Traditional IP addressing system.
- Most common today.

Example:

- 192.168.1.10

---

### IPv6

- Newer IP addressing system.
- Much larger address space.

---

### CLI (Command-Line Interface)

- Text-based management tool.
- Used to manage Azure resources through commands.

Example:

- Azure CLI

---

### API (Application Programming Interface)

- Allows applications to communicate.
- Used for automation and integrations.

Think:

- Software talking to software.

---

### IP (Internet Protocol)

- Address used to identify devices on a network.

Examples:

- Public IP
- Private IP

---
